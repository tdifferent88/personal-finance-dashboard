# Dashboard de Gastos Pessoais

Dashboard interativo feito com Dash + Plotly + Pandas para análise financeira pessoal.

## Features

- Upload/Leitura de CSV
- Gastos mensais
- Gastos por categoria
- Dashboard interativo

## Stack

- Python
- Pandas
- Plotly
- Dash

## Como rodar

```bash
pip install -r requirements.txt
python src/app.py
```
